# Lessons — Cross-Project Craft Notes

Cross-project lessons distilled from analysis runs. NOT script-specific staging — that lives in `directors_notes.md` per project. This file holds principles you want to carry into the next script.

Two categories:

1. **Story craft** — what you've learned about writing that applies to all your work
2. **Pipeline craft** — what you've learned about running this analysis (process improvements, prompts that work, anti-patterns)

Add entries as you finish each project. Date each entry. Cite the project and skill that surfaced the lesson.
