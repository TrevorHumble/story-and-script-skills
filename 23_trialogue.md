# Skill 23 — Trialogue Technique

## Purpose
A trialogue is a scene where two characters use a THIRD ELEMENT — an object,
topic, person, or activity — as the arena for a conflict they cannot address
directly. Instead of fighting about the real subject, they fight about the canaries.
The third element carries the subtext so the characters don't have to.

This is both a diagnostic and a rewrite tool. It diagnoses scenes of emotional
confrontation that are failing because the characters speak too directly — and
provides a structural solution.

The greater the distance between the third element and the true subject,
the stronger the subtext.

## Inputs
- script.txt
- 20_on_the_nose.md
- 22_said_unsaid.md

## Instructions

If a required input file is missing or does not contain the expected content, halt immediately and report the missing input to the orchestrator — do not proceed with partial context.

### Step 1 — Identify Candidate Scenes
Using the on-the-nose diagnosis and inner life map: find scenes where two
characters are in genuine emotional conflict but the current dialogue is either
too direct (on the nose) or too vague (the conflict isn't landing).

These are the scenes that could benefit from the trialogue technique.

### Step 2 — Name the True Subject
For each candidate scene: what is the scene ACTUALLY about?
The real emotional conflict, the real power dynamic, the real fear or desire
at the center of it.

### Step 3 — Find a Third Element
What object, topic, activity, or absent third party could both characters
have a relationship to — and could serve as the displacement target for the
real conflict?

The best third elements:
- Both characters can talk about it naturally without it feeling forced
- It has qualities that MIRROR or CONTRAST the real subject
- Using it as a proxy produces irony, tension, or humor

### Step 4 — Test the Trialogue
Sketch (in notes, not full rewrite) how the scene would work if both characters
spoke only about the third element. Does the real conflict come through?
If you removed the third element and had them speak directly, would the scene
be MORE or LESS powerful? If more powerful: the trialogue isn't working.
If less powerful: it's working.

### Step 5 — Assess the Existing Script
Does the script already use trialogue successfully anywhere?
Identify instances where a third element is already carrying subtext well,
and instances where one could be introduced.
