# Skill 21 — Exposition as Ammunition

## Purpose
Exposition — backstory, context, worldbuilding, relationship history — is not the
enemy. Exposition delivered AS EXPLANATION is the enemy. McKee's technique: have
characters use information about the past as ammunition in their current conflicts.
When a character reveals something because they NEED IT TO WIN RIGHT NOW, the
exposition disappears into the action and the audience absorbs it without noticing.

McKee's paradigm: Darth Vader's "I am your father" is delivered at the moment
of maximum conflict — as a weapon to disarm his son. Not as backstory. As ammunition.

## Inputs
- script.txt
- 19_beat_analysis.md
- 20_on_the_nose.md

## Instructions

If a required input file is missing or does not contain the expected content, halt immediately and report the missing input to the orchestrator — do not proceed with partial context.

### Step 1 — Inventory All Exposition
Read the script and mark every passage of exposition: backstory, world explanation,
relationship history, context-setting. Include both dialogue and narrative description.

### Step 2 — Test Each Passage
For each marked passage: is this information delivered because a character needs
it as a weapon or tool in the current conflict — or because the audience needs to
know it?

If for the audience: this is expository telling. It must be moved or transformed.

### Step 3 — Apply the Withholding Principle
McKee: withhold exposition unless it is absolutely necessary for comprehension.
The least important facts come early. Critical secrets come last.
Is the script revealing things in the right order? Or is it giving away information
that would be more powerful held until later?

### Step 4 — Find the Ammunition Moment
For each piece of exposition that needs to be moved: identify a moment of conflict
elsewhere in the script where this information would function as ammunition —
where a character would USE this knowledge to get what they want right now.

### Step 5 — Check the "As You Know Bob" Lines
Cross-reference with Exercise 20's exposition problems. Are characters still
telling each other things they both already know? These must be resolved.

## Output Format

If the script does not provide enough material to complete this analysis confidently, mark the output [INCONCLUSIVE] at the top of the file, describe what is missing or ambiguous, and flag it for human review before the pipeline continues.
