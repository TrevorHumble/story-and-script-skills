# Skill 06 — Inciting Incident Analysis

## Purpose
The inciting incident is not just the first event — it is the event that changes
everything. It must radically upset the balance of the protagonist's life, create
a desire they must now pursue, and generate a question the audience cannot stop
wondering about until the climax answers it.

A weak or delayed inciting incident is one of the most common structural failures.
The story has nowhere to go if it doesn't know what it's launched.

## Inputs
- script.txt
- beat_sheet.txt
- 02_controlling_idea.md
- 05_act_structure.md

## Instructions

If a required input file is missing or does not contain the expected content, halt immediately and report the missing input to the orchestrator — do not proceed with partial context.

### Step 1 — Identify the Inciting Incident
Name the specific event. Not a mood shift — a concrete, dramatizable occurrence. Measure it against the controlling idea: does this event force the protagonist into the story's central value conflict, or is it unconnected to what the story ultimately argues?

### Step 2 — Test for Radical Upset
Does this event RADICALLY upset the protagonist's life, or merely inconvenience it?
A moderate upset produces a moderate story. The test: could the protagonist
reasonably ignore this event and go back to their normal life? If yes, the
inciting incident is too weak.

### Step 3 — Test for Desire
Does this event cause the protagonist to form a desire — conscious or unconscious —
that they must now pursue? What is that desire? Does it match the want
identified in the Story Spine?

### Step 4 — Test for Story Question
Does this event create a question the audience will be asking until the climax?
Write the question. Is it compelling enough to sustain the story?

### Step 5 — Check Placement
When does the inciting incident occur? For a short film, it should arrive early —
often in the first scene. If it's delayed: what is the story doing before it arrives?
Is that pre-inciting material earning its place, or is the story spending time
it doesn't have?

### Step 6 — Check Causality
Is the inciting incident CAUSED by something — a real event in the story's world —
or is it a character's internal mood change? McKee: the inciting incident must be
dramatizable. It happens in the world.
