# Skill 04 — Image System Exercise
# 📋 ONE-PAGER SUMMARY AFTER THIS EXERCISE

## Purpose
An image system is a recurring motif — a category of images, objects, colors,
sounds, or visual patterns — that accumulates meaning across the story beneath
the level of plot. It operates subliminally: the audience absorbs it without
being told to, and it deepens the emotional and thematic resonance of the story.

This runs early because in some stories, the image system IS the story — it
carries meaning that the plot alone cannot. Identifying it early means later
exercises can check whether scenes are contributing to it or working against it.

McKee's test: remove the image system and see if the story loses something real.

## Inputs
- script.txt
- 02_controlling_idea.md
- 03_story_spine.md

## Instructions

If a required input file is missing or does not contain the expected content, halt immediately and report the missing input to the orchestrator — do not proceed with partial context.

### Step 1 — Inventory Recurring Elements
Read the script and list every element that appears more than once:
- Visual objects (the box, the gum, the pink color, specific settings)
- Colors or color contrasts (saturated vs. desaturated)
- Sounds or music
- Physical actions or gestures
- Textures or materials
- Animals or natural elements
- Technology or media

Do not filter. List everything that repeats.

### Step 2 — Group Into Systems
Cluster the recurring elements into categories. Look for patterns
you may not have expected. Name each cluster.

### Step 3 — Test for Accumulated Meaning
For each cluster: does the meaning of this element deepen or shift
each time it recurs? Or does it just repeat without development?
Accumulated meaning = image system.
Repetition without development = decoration.

### Step 4 — Identify the Dominant System
Which cluster carries the most thematic or emotional weight?
This is the dominant image system. Name it and describe what it is doing.

### Step 5 — Connect to Controlling Idea and Spine
Does the dominant image system connect to the controlling idea?
Does it track the protagonist's spine — does it shift or develop
in parallel with the protagonist's journey?
If the image system and the controlling idea are pulling in different
directions, the story may be telling two different stories at once.

### Step 6 — The Removal Test
Pick two or three key scenes. Mentally strip the dominant image system
from them. Does the scene lose resonance? Does it feel thinner?
If yes: the image system is working.
If no: it may be present but not load-bearing.

### Step 7 — Check for Involuntary Systems
Are there patterns that appeared without conscious intention that are
pointing the story somewhere surprising? These are often more true to
the story's real meaning than the consciously chosen ones.
Note any you find.
